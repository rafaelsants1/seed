# SEED - Sistema de Avaliação da Secretaria da Educação

Projeto acadêmico de uma plataforma institucional para gestão de provas, simulados, notas, desempenho e banco de questões.

## Estrutura

- `frontend/` — interface web em React + TypeScript + Tailwind
- `backend/` — futura API e regras de negócio
- `docs/` — documentação funcional e técnica

## Perfis previstos

- Aluno
- Professor
- Secretaria

## Funcionalidades previstas

- Login e cadastro
- Provas e simulados
- Lançamento de notas
- Local de prova
- Histórico
- Desempenho
- Banco de questões
- Relatórios

## Tecnologias do frontend

- React
- TypeScript
- Tailwind CSS
- React Router
- Recharts
